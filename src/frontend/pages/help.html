Help & Instructions

Getting Started

  1. Take pictures - Press the big DESCRIBE button or use Alt+D
  2. Listen - Augen will speak the description out loud
  3. That's it! - No setup required, just start using

Best Practices

  • Good lighting - Take pictures in well-lit areas when possible
  • Steady hands - Keep the camera as still as possible
  • Close for text - Move closer to read menus, signs, or documents
  • Be patient - Processing can take a few seconds

Troubleshooting

Camera not working?

  • Make sure you allowed camera permissions
  • Try refreshing the page
  • Check if other apps can use your camera

No sound?

  • Check your device volume
  • Make sure audio is not muted
  • Try using headphones

Service issues?

  • Try refreshing the page
  • Check your internet connection
  • Wait a moment and try again

Browser Support

Augen works best on:

  • Mobile - Chrome, Safari, Firefox on iOS/Android
  • Desktop - Chrome, Firefox, Edge, Safari
