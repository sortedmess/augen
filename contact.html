Feedback & Suggestions

We'd love to hear from you! Your feedback helps make Augen better for everyone.

Report Issues

If you encounter problems or bugs, please let us know with details about:

  • Device - What phone/tablet/computer you're using
  • Browser - Chrome, Safari, Firefox, etc.
  • Problem - What happened vs. what you expected
  • Steps - How to reproduce the issue

Accessibility Feedback

Accessibility is our priority. If you have trouble using Augen with:

  • Screen readers - NVDA, JAWS, VoiceOver, TalkBack
  • Voice control - Dragon, Voice Control, Voice Access
  • Switch navigation - Physical switches or assistive devices
  • Motor limitations - Difficulty with touch or clicks

Please share your experience so we can improve.

Technical Information

Augen is powered by advanced AI technology that provides:

  • Image recognition - Identifies objects, text, and scenes
  • Text reading - Reads signs, menus, and documents aloud
  • Natural speech - High-quality text-to-speech output
  • Real-time processing - Fast analysis and response

Privacy & Security

  • No personal data stored - Images processed temporarily only
  • No tracking - We don't collect usage data
  • Secure processing - Images handled safely and securely
  • Open source - Code is transparent and auditable
